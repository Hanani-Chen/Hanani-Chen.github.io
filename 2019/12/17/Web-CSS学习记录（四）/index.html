Web-CSS学习记录（四）

Posted on 2019-12-17

盒状模型Box Model

盒状模型图

  • 元素的背景（background）是填充在由内容和内边距、边框组成的区域

margin & padding

属性简写规则

  • 一个值=所有边距

  • 两个值=上下边距+左右边距

  • 三个值=上边距+左右边距+下边距

  • 四个值=从上边距开始顺时针

  • 简写属性的值之间使用空格隔开

边框border

  • 边框可以应用到在主体元素body 中的绝大部分元素上

  • border-style 依旧可以类似边距简写

继承属性的例外

  • 可用inherit允许继承

尺寸、定位，布局和显示属性

Dimension 尺寸

  • 即通过拉动浏览器页面大小所呈现的样式

  • 即流动布局与固定布局

定位属性position

static

  • p、h1、div 等元素称为块元素，显示为一块内容
  • strong 和span 等元素称为内联元素，显示在行内
  • 在文档流中，块级元素从上到下一个接一个排列；内联元素在一行中水平布置

relative

absolute

  • 自己的原本的位置不保留

fixed

  • 生成绝对定位的元素，相对于浏览器窗口进行定位。

  • 元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。

  • 坐标系会根据所选的位置属性组合进行变换

  • 叠字火狐好像会自动换行

z-index

布局属性

float浮动

  • 当把bian加入float:left,并打上标记
  • box2被覆盖

clear清除

  • 似乎是后浮动的才能清除

显示属性

display属性

visibility 属性
