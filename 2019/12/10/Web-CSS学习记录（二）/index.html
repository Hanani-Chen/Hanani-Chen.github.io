字体样式

字体族与字体

字体族 实例
Serif 有衬线 Tiomes New Ro man, Garamond
Sans-serif 无衬线 MS Arial, Helvetica
Monospace 等宽字体 Caflisch Script, Zapf-Chancery
Cursive 草书体 Critter, Cottonwood
Fantasy 幻想体 Courier, Prestige
  • 有无衬线区别

  • 属性font-family: Tahoma, SimSun, Sans-Serif; 按照从左往右使用第一个存在的字体

关键字与百分比

  • 子元素定义的属性与父元素重复时，子元素属性覆盖父元素属性
  • %是在父元素基础上改变

字体加粗

  • 属性名font-weight

其他字体属性

  • 字体变种 font-variant
    • normal（缺省值，正常显示）
    • small-caps（以小型大写字母方式显示）
  • 字体风格 font-style
    • normal（正常）
    • italic（意大利体）
    • oblique（倾斜体）
