gcc命令

-o

  • gcc hello.c -o hello 编译生成执行文件
  • gcc -E/S/c hello.c -o hello.i/s/o 信息输出到文件中

-c

  • gcc -c hello.c -0 hello.o 生产编译文件
  • gcc -c functions/greeting.c

-g

  • gcc hello.c -g -o hello 生成调试信息供gdb所需

-L/l(大写)

  • l是链接到库文件名
  • L是链接到库所在目录

-i（大写）

  • 用来链接头文件的目录
